# 💬 Chatters — Social App

A multi-language social media web app with friend code system.

## 🌍 Features
- 18+ Languages supported
- Friend Request system with 7-digit code
- Stories, Posts, Reels
- Profile page
- Permission screen
- Phone & Google Login

## 🚀 Live Demo
Visit the live website after deploying on GitHub Pages.

## 🛠️ Tech
Pure HTML, CSS, JavaScript — No frameworks needed!
